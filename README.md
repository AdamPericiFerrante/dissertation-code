# Empirical Study: Individual-Based Approach to applying modified EWMA Control Charts on USD/EUR Data

This folder contains the R code for the empirical study in the dissertation. The analysis applies EWMA-type control charts on an individual basis to USD/EUR exchange rate data under non-normal conditions.

## Overview

The empirical study focuses on:

- Transforming exchange rate data into log-returns  
- Testing for stationarity  
- Fitting non-normal distributions  
- Estimating skewness and kurtosis  
- Applying EWMA-based control charts  

## Data Requirments 

Find attached in this repository the data set used in this dissertation which contains the USD/EUR spot exchange rate within the period (2015-2025) interpolated for non-trading days, as well as other variables considered for future multivariate studies. 

Please download the CSV file and import it into your R workspace.
